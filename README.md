# Gearup - E-commerce Bicycle Shop

## Description

Gearup is an e-commerce platform dedicated to selling bicycles. Whether you're a casual rider or a professional cyclist, Gearup offers a wide range of bicycles to suit your needs.

## Technologies Used

- React
- Redux
- TypeScript
- Stripe

## Features

- Browse a variety of bicycles categorized by type, brand, or price range.
- View detailed product descriptions, images, and specifications.
- Add items to your cart and proceed to checkout.
- Secure payments through Stripe integration.
- User authentication and account management (optional feature).

## Usage

- Visit [Gearup](hhttps://gearup-bice.vercel.app) to explore and purchase bicycles.
- Enjoy a seamless shopping experience with intuitive navigation and secure transactions.
